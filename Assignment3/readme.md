# Assignment 3: Text Cleaning, Stopwords Removal, Stemming and Lemmatization

## Problem Statement
Implement text preprocessing techniques including text cleaning, stopwords removal, stemming and lemmatization using Python.

## Tools Used
Python
Jupyter Notebook
NLTK

## Description
Text cleaning removes punctuation and special characters from text. Stopwords removal eliminates commonly used words that do not contribute to meaning. Stemming reduces words to their root form, while lemmatization converts words into meaningful base forms.

## Conclusion
Text preprocessing techniques are essential for preparing textual data for Natural Language Processing applications.
