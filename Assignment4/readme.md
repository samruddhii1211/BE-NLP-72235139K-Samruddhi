
# Assignment 4: Transformer Model

## Problem Statement
Implement a Transformer model using the PyTorch library.

## Tools Used
Python  
Jupyter Notebook  
PyTorch  

## Description
Transformers are deep learning models used in Natural Language Processing. They use attention mechanisms to process sequences of text efficiently.

In this assignment a simple transformer model was implemented using the PyTorch nn.Transformer module.

## Conclusion
Transformer models are widely used in modern NLP applications such as translation, chatbots and text generation.
