# Assignment 2: Bag of Words, TF-IDF and Word2Vec

## Problem Statement
Implement text vectorization techniques using Bag of Words, TF-IDF and Word2Vec.

## Tools Used
Python
Jupyter Notebook
scikit-learn
gensim

## Description
Bag of Words converts text into frequency-based vectors.

TF-IDF assigns weights to words based on their frequency and importance.

Word2Vec generates vector representations capturing semantic relationships between words.

## Conclusion
Text vectorization techniques help convert text data into numerical form for machine learning models in NLP.
